# Personal Portfolio

A professional and interactive personal portfolio web application designed to showcase projects, skills, and professional experience. Built with a focus on modern user experience, responsive design, and dynamic data integration.

## Features

- **Responsive Design:** Optimized layout for seamless viewing across desktop, tablet, and mobile devices.
- **Dynamic Content:** Integration with Supabase for real-time content management of projects, skills, and profile information.
- **Interactive UI Elements:** Advanced CSS animations, scroll reveals, and device orientation tracking (gyroscope) for mobile interactivity.
- **Admin Dashboard:** Secure authentication and management interface for updating portfolio content directly from the application.

## Installation and Setup

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Configure environment variables for Supabase connection (`VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`).
4. Run the development server using `npm run dev`.

## Architecture

The application follows a modular component-based architecture using React. Global styling and animations are managed through standard CSS, maintaining high performance and clean separation of concerns. State management utilizes React hooks, ensuring efficient rendering and data flow.
